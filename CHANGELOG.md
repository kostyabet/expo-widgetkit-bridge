## [0.1.0] — 2026-07-08

### Added
- Initial release.
- `reloadAllTimelines()` — reload every WidgetKit timeline in the app.
- `reloadTimelines(kind)` — reload a single widget kind.
- `getCurrentConfigurations()` — enumerate the widgets the user has installed.
